Getting Started
✅ Your Anthropic API key never leaves your machine. Token Manager only receives token counts, model name, and cost — zero credentials.
Step 1 — Set your Anthropic key in your environment
# Mac / Linux
export ANTHROPIC_API_KEY=sk-ant-your-key

# Windows PowerShell
$env:ANTHROPIC_API_KEY="sk-ant-your-key"
Your Anthropic key stays here. Token Manager never sees it.
Step 2 — Install the SDK
pip install anthropic
